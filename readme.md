# Responsive Landing Page - SkillCraft Technology

This repository contains the source code for **Task 01** of the Web Development Internship at [SkillCraft Technology](link-to-skillcraft-website-or-linkedin). 

## 📝 Project Overview

The objective of this task was to build a responsive landing page with a strong focus on the user interface and navigation experience. The main feature of this project is a dynamic, interactive navigation menu designed to improve site aesthetics and usability.

## ✨ Key Features

* **Interactive Navigation Menu:** The menu items change color and style when hovered over, providing immediate visual feedback to the user.
* **Dynamic Scroll Effects:** The navigation bar intelligently alters its appearance (e.g., background color, opacity, or shadow) when the user scrolls down the page.
* **Fixed Positioning:** The navigation menu remains anchored to the top of the viewport, ensuring it is always visible and accessible regardless of where the user is on the page.
* **Fully Responsive:** The layout adapts seamlessly to different screen sizes, ensuring a great experience on desktops, tablets, and mobile devices.

## 🛠️ Tech Stack

* **HTML5:** For structuring the landing page content.
* **CSS3:** For styling, responsive design (media queries), and handling hover/scroll transitions.
* **JavaScript:** To detect scroll events and dynamically toggle CSS classes for the navigation bar.
